# Vue

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```

# Json Server

### Project setup

```
npm install -g json-server
```

### Start Json server

```
json-server --watch db.json

```
